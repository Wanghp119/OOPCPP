# 实习（五）数组、字符串和指针
## 实习目标
掌握数组、字符串和指针的用法。
- string表示可变字符串，它的初始化方式分为两种：直接初始化和拷贝初始化，以及如何处理string中的单个字符。
- 数组是若干个相同类型变量的集合，在存储空间上连续。掌握数组定义、初始化、访问以及动态分配数组单元
- 指针是C语言中极其重要的部分，在C++中，迭代器和应用可以替代指针的部分功能，但灵活的指针在很多场合还是会用到，掌握各种类型的指针用法，是深入学习C++的基础。
